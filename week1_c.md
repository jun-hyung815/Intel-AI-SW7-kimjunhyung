## 학습 내용 정리

### 조건문
- `if ~ else`, 중첩 조건문, `switch`문 구조 이해


## 실습 예제

### 구구단 출력
```c
for (int i = 2; i <= 9; i++) {
    for (int j = 1; j <= 9; j++) {
        printf("%d x %d = %d\n", i, j, i*j);
    }
}
```

### 최대공약수 (GCD)
```c
int gcd(int a, int b) {
    while (b != 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}
```

### 평균값 구하기
```c
int scores[5] = {80, 70, 90, 60, 100};
int sum = 0;
for (int i = 0; i < 5; i++) {
    sum += scores[i];
}
printf("평균: %.2f", (float)sum / 5);
```
